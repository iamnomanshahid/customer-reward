

All Login present in app/services/rewards

### Requirements:
 - `Ruby version >= 2.3.1`
 - `Bundler version 1.12.5`

### Install:
 - `bundle install`

### Tests:
 - `bundle exec rubocop`
 - `bundle exec rspec`

### Run:
 - `rails s`
 - `open new terminal and run command`
 - `curl -X POST localhost:3000/rewards --data-binary @input.txt -H "Content-Type: text/plain"`
